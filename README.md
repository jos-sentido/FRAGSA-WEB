# FRAGSA — Sitio oficial

Sitio web del grupo constructor **FRAGSA (Grupo Integral)**. Proyecto basado en React + Vite + Tailwind (CDN).

## Documentos de marca
- [`BRAND_GUIDELINE.md`](./BRAND_GUIDELINE.md) — paleta, tipografía, sistema visual, componentes UI.
- [`SITE_CONTENT.md`](./SITE_CONTENT.md) — arquitectura de información y copy real por sección.

## Assets de marca

- `logo-fragsa-color.png` · `logo-fragsa-mono-dark.png` · `logo-fragsa-white.png`
- `glifo-fragsa.png` · `glifo-fragsa-navy.png` · `glifo-fragsa-white.png`
- `favicon.ico` · `favicon-16.png` · `favicon-32.png` · `favicon-64.png`

## Correr local
```bash
npm install
npm run dev
```
Abre http://localhost:8080.

## Build
```bash
npm run build
npm run preview
```
